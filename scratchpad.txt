# DeepAgent: Port SkyworkAI DeepResearchAgent to Orbiter

## Context

The existing Jina DeepSearch implementation at `examples/advanced/deepsearch/` has unusable
performance. We're replacing it with a port of SkyworkAI's DeepResearchAgent — #1 on the GAIA
benchmark (82.42%), beating OpenAI Deep Research and Manus. Source is cloned at
`/tmp/DeepResearchAgent/`.



## Architecture: Key Insight

Orbiter's `Swarm(mode="team")` already provides exactly the orchestration that SkyworkAI
implements manually with PlanningAgent + AgentBus + ToolCallingAgent:

**SkyworkAI:** `PlanningAgent → PlanDecision{dispatches} → AgentBus → ToolCallingAgent → results
back`
**Orbiter:** `Swarm(team) → Lead Agent calls delegate_to_researcher(task="...") → Worker runs →
result returned as tool output → Lead continues`

The `_DelegateTool` in `swarm.py:918-966` wraps each worker as a tool. This eliminates
PlanDecision structured output, AgentBus, and ToolCallingAgent entirely.

class DeepAgentConfig(BaseModel):
    lead_model: str = "openai:gpt-4o"
    researcher_model: str = "openai:gpt-4o-mini"
    tool_model: str = "openai:gpt-4o-mini"       # For internal tool LLM calls

    search_num_results: int = 5

    content_reader: str = "httpx"                  # httpx | jina

search
    lead_max_steps: int = 15
    researcher_max_steps: int = 10
    brave_api_key: str | None = None
    serper_api_key: str | None = None
    jina_api_key: str | None = None
    output_dir: str = "deepagent_output"

    @classmethod
    def from_env(cls) -> DeepAgentConfig: ...

### 2. `tools/web_search.py` — Tool subclass, multi-backend

- `WebSearchTool(Tool)` with `execute(query, num_results=5) -> str`
- Backends: `_search_duckduckgo()` (free, `DDGS().text()` via `asyncio.to_thread()`),
`_search_brave()`, `_search_serper()`, `_search_jina()` (all via httpx)
- Returns formatted string: title, URL, snippet per result

### 3. `tools/content_reader.py` — Tool subclass

- `ContentReaderTool(Tool)` with `execute(url) -> str`
- Default: httpx + beautifulsoup4, 20s timeout, max_length truncation
- Optional: Jina Reader API when configured

### 4. `tools/deep_researcher.py` — Core research logic (port of SkyworkAI DeepResearcherTool)

- `DeepResearcherTool(Tool)` with `execute(task, max_rounds=3) -> str`
- **Internal multi-round loop** (ported from
`/tmp/DeepResearchAgent/src/tool/workflow_tools/deep_researcher.py`):
  1. Generate search query via internal LLM call (`_generate_search_query()`)
  2. Parallel search: web_search + optional LLM search models (`_parallel_search()`)
  3. Merge results (`_merge_search_results()`)
  4. Evaluate completeness via structured LLM output (`_evaluate_completeness()`)
  5. If complete → break, else → next round
  6. Generate final summary (`_generate_summary()`)
- Gets its own `ModelProvider` via `get_provider(tool_model)` in constructor for internal LLM
calls
- Uses `CompletenessEvaluation(BaseModel)` Pydantic model for structured eval

### 5. `tools/file_analyzer.py` — Simplified DeepAnalyzerTool port

- `FileAnalyzerTool(Tool)` with `execute(task, file_path) -> str`
- Reads local files or fetches URLs, returns content summary

### 6. `prompts.py` — System prompts

- `LEAD_PROMPT`: Planning agent that delegates to workers via `delegate_to_researcher(task=...)`
and `delegate_to_analyzer(task=...)`, synthesizes results
- `RESEARCHER_PROMPT`: Research specialist with deep_research, web_search, read_webpage tools
- `ANALYZER_PROMPT`: File analysis specialist

### 7. `agents.py` — Assembly

```python
def build_deep_agent(config: DeepAgentConfig | None = None) -> Swarm:
    cfg = config or DeepAgentConfig.from_env()

    # Tools
    web_search = WebSearchTool(provider=cfg.search_provider, ...)
    content_reader = ContentReaderTool(mode=cfg.content_reader, ...)
    deep_researcher = DeepResearcherTool(tool_model=cfg.tool_model, web_search=web_search, ...)
    file_analyzer = FileAnalyzerTool(...)

    # Workers
    researcher = Agent(name="researcher", model=cfg.researcher_model,
                       instructions=RESEARCHER_PROMPT,
                       tools=[deep_researcher, web_search, content_reader],
                       max_steps=cfg.researcher_max_steps)
    analyzer = Agent(name="analyzer", model=cfg.researcher_model,
                     instructions=ANALYZER_PROMPT,
                     tools=[file_analyzer, content_reader],
                     max_steps=cfg.researcher_max_steps)

    # Lead (Swarm auto-adds delegate_to_researcher, delegate_to_analyzer)
    lead = Agent(name="planner", model=cfg.lead_model,
                 instructions=LEAD_PROMPT, max_steps=cfg.lead_max_steps)

    return Swarm(agents=[lead, researcher, analyzer], mode="team")
```
